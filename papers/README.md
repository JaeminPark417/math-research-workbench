# Papers

This folder contains reading notes, not the paper files themselves. Create one
Markdown note per paper, preprint, book, or other citable source using
`meta/templates/paper.md`.

Before marking a citation verified, compare its title, authors, year, and
identifier with an authoritative record such as the publisher, DOI record, or
arXiv abstract page. If anything remains uncertain, leave it blank and keep
`citation_status: unverified`. Never ask Codex to complete a citation from
memory.

In the note, separate:

- what the source explicitly states;
- your own interpretation;
- possible extensions suggested by you or an assistant.

PDFs and other binaries live outside the Git-backed workspace by default. Put a
safe shared-drive URL or user-configured portable reference in `file_ref`; do
not store credentials or private sharing tokens.
