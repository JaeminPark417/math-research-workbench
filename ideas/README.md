# Ideas

This folder holds mathematical questions, conjectural directions, and promising
fragments that have been clarified beyond the inbox stage.

Use one file per idea, named in lowercase `kebab-case`, for example
`equidistribution-under-weak-mixing.md`. Start from
`meta/templates/idea.md`.

An idea should make clear:

- the exact mathematical question;
- definitions, hypotheses, and quantifiers;
- examples and boundary cases;
- supporting evidence and counterevidence;
- known obstacles and open questions;
- links to relevant papers and projects.

Ideas evolve. Update the evolution log when the formulation changes. Move
failed approaches to the archived section instead of deleting them. Change the
frontmatter status only to one of the values in `meta/schemas.md`.
